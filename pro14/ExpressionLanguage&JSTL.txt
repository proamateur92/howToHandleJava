JSP 발전 과정
초기에는 html 태그를 중심으로 자바를 이용해 화면을 구현 -> 화면 요구사항이 복잡해지면서 액션태그 등장
-> JSP2.0에서는 페이지 안에서 복잡한 자바 코드를 제거하는 쪽으로 발전(디자이너 입장에서 JSP 안에 복잡한 자바 코드가 있어 화면 작업이 어려웠기 때문)
-> 현재 JSP 페이지는 표현언어와 JSTL을 사용하여 구현

표현 언어 특징
1. 기존 표현식보다 편리하게 값 출력
2. 변수와 여러가지 연산자 포함
3. JSP 내장 객체에 저장된 속성 및 자바의 빈 속성도 표현 언어에서 출력
4. 표현 언어 자체 내장 객체 제공
5. JSP 페이지 생성 시 기본 설정은 표현 언어를 사용할 수 없음
6. 페이지 디렉티브 태그에서 반드시 isELIgnored=false로 설정해야 한다.
